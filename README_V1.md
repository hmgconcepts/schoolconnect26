# School Connect v1

School Connect v1 is a corrected, deployment-ready generator package for building free school management portals using static files and Supabase.

## What this package contains
- generator pages
- assets
- SQL schema files
- PWA files
- deployment docs
- audit docs
- QA checklist
- role permission matrix
- detailed features guide

## Primary improvements targeted in this v1 package
- clearer role/dashboard separation
- safer permission expectations
- more accurate documentation
- better deployment clarity
- no paid AI API dependency

## Important
This release follows a safe verification path:
- source-level and packaging-level issues were audited directly
- items requiring live authenticated database interaction are documented for QA rather than falsely declared fixed without runtime proof
