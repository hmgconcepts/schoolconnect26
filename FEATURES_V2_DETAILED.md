# School Connect v2 — Detailed Features Guide

School Connect v2 is a free, modular school portal generator that builds static school sites connected to Supabase for authentication, database, permissions, and school operations.

## What v2 improves
v2 specifically improves practical role usability after deployed-site feedback:
- admins can now manage operational modules more appropriately
- staff can reach academic tools they need for daily work
- parents can read child-linked academic and financial pages
- students can read self-facing academic and communication pages

## Major feature groups

## 1. Generator and branding
- browser-based wizard
- branding controls
- module selection
- downloadable ZIP output
- static deployment friendliness

## 2. Role-based school portal
### Admin / Super Admin
Can oversee and manage setup, governance, academics, finance, HR, records, operations, and reporting.

### Staff / Teacher
Can manage academic work such as attendance, results, CBT, report cards, lesson plans, assignments, and related operational workflows.

### Parent
Can view child-linked information such as results, fees, report cards, assignments, attendance, timetable, announcements, complaints, and messages.

### Student
Can view self-facing information such as assignments, results, report cards, timetable, resources, certificates, messages, complaints, and announcements.
